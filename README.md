# VILMA Project

#### Packages
* **[vilma_framework]():** Catkin metapackage for VILMA Project.
* **[vilma_core]():** Base package of the VILMA Project.
* **[vilma_env_toolchain]():** Environment toolchain package to provide tools to generate a new project from template and other stuff.
* **[vilma_task_manager]():** Base class to the envent driven process.
* **[vilma_hartbeat]():** Base class to the periodic process. 

**See each package's README.md file for more information.
